C# Extension Methods

Helper class to provide some convenient Extension Methods for your projects.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the C# Extension Methods page.

This is just a simple project to provide a place for common or convenient (often-used) extension methods. It is also a nice project to use to illustrate the use of Extension Methods.

Documentation

The .chm file provided in the []Documentation folder will be updated to include new additions to the Extension methods as time goes on.
